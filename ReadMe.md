# AWS Web Server Deployment Project

This project demonstrates how to deploy and configure web servers on AWS using EC2 instances.

## Technologies Used
- Amazon EC2
- Windows Server
- Linux (Amazon Linux / Ubuntu)
- IIS Web Server
- Apache Web Server
- Elastic IP
- SFTP

## Tasks Implemented

### Task 1
Create a Windows EC2 instance in AWS.

### Task 2
Configure IIS Web Server on Windows Server.

### Task 3
Configure Elastic IP address for Windows Web Server.

### Task 4
Create a Linux EC2 instance in AWS.

### Task 5
Configure Apache Web Server on Linux.

### Task 6
Attach Elastic IP to the Linux server.

### Task 7
Configure SFTP access to Linux Web Server.

## Project Outcome
Successfully deployed both Windows and Linux web servers on AWS and configured remote access and static IP.
